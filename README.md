# Özkan Kaya Portfolio

Personal portfolio of **Özkan Kaya**, a Computer Engineer and Full-Stack Developer based in Istanbul, Türkiye.

> [!NOTE]
> This portfolio is actively under development. The README will be updated with complete documentation when the project is finished.

## Current Features

- Responsive, terminal-inspired About page
- Procedurally generated animated ASCII galaxy
- Text scramble and scroll-driven transition effects
- Adaptive sticky navbar with light and dark themes
- Resume, GitHub, and LinkedIn links
- SSR and reduced-motion support

## Tech Stack

- Angular 22 and TypeScript 6
- Tailwind CSS 4
- Angular SSR and Express 5
- Vitest 4
- Node.js 20.x and npm 11.x

## Development

```bash
npm install
npm start
```

Open [http://localhost:4200](http://localhost:4200).

Production build:

```bash
npm run build
npm run serve:ssr:ozkankaya-portfolio
```
